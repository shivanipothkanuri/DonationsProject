# User Documentation

## Roles and Capabilities

- Donor: register/login, donate, recurring donations, download receipt.
- Staff: manage campaigns, beneficiaries, inventory, aid requests.
- Case Manager: review, approve, deny, escalate requests.
- Leadership: view KPI and strategic reports.
- Administrator: full user management, audits, templates, compliance reports.

## Core Flows

1. Account setup and login
2. Campaign browsing and donation
3. Beneficiary registration and aid request lifecycle
4. Reporting and exports
5. Administrative governance actions

## Support Notes

- Locked accounts can be unlocked by Administrator.
- Password reset uses token email workflow.
- MFA is available for Administrator accounts.
